1) Посмотреть примеры, прочитать (и понять!) EADME.md, а также комментарии в коде
2) Придумать предметную область, которую можно описать набором иерархических классов
должен быть один базовый класс, обобщающий объекты предметной области и несколько производных.
Например базовый класс - Автомобиль, который содержит методы и свойства, характерные для всех авто,
и несколько производных - типа Кабриолет, Джип, Грузовик.
У базового класса должно быть 3 свойства и один метод.
У производных - 3 доп свойства, один свой внутренний (вспомогательный) метод и один перегруженный.
3) Реализовать данные классы с примерами их использования
4) Использовать для этого online IDE - https://ideone.com/ (все в одном файле - без разбиения на h/cpp)
5) Скопировать ссылку вида https://ideone.com/IWfqw8, где IWfqw8 - будет уникальной строкой
6) в разделе Issues данной репы разместить запись с данной ссылкой:
  в Заголовке (Titles) написать Фамилию, Инициалы - Номер группы
  в Leave a comment разместить саму ссылку.
